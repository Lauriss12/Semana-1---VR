Durante este curso aprendí que la tecnología puede ser una herramienta importante para fortalecer la participación ciudadana y la organización comunitaria. También comprendí la importancia de utilizar bases de datos para almacenar información de manera ordenada y segura. Aprendí que la autenticación protege el acceso a la plataforma y ayuda a cuidar la privacidad de los usuarios. Además, conocí el uso de automatizaciones para facilitar tareas repetitivas y mejorar la comunicación. El uso de herramientas como Git y GitHub permitió mantener un control de versiones del proyecto. Igualmente, entendí la importancia de trabajar con datos simulados durante las prácticas. Todos estos conocimientos contribuyen al desarrollo de soluciones tecnológicas más responsables, organizadas y útiles para la comunidad.
