Sistema operativo:
Es el software principal que administra los recursos del equipo. Permite ejecutar programas y controlar el funcionamiento general del sistema.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear aplicaciones y servidores.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y las respuestas que reciben los usuarios.

Puerto:
Es un punto de comunicación que permite recibir solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad y eventos del servidor. Ayudan a identificar errores y monitorear el funcionamiento.
