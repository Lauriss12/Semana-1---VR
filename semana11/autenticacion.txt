El login permite identificar a un usuario antes de acceder a las funciones de una aplicación. La autenticación ayuda a verificar que la persona sea quien dice ser. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes. Solo los usuarios autorizados deben acceder a estas funciones. No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves API ni otros datos sensibles de la comunidad.
